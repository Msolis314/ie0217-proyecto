Descripción detallada

Clase que representa los intereses de un Certificado de Depósito a Plazo.

Esta clase es abstracta y tiene como objetivo calcular los intereses de un CDP.

Esta clase tiene como atributos un CDPData, un plazo, un capital, una moneda y un periodo.
