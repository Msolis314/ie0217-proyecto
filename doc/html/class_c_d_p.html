Descripción detallada

Clase que representa un Certificado de Depósito a Plazo.

Esta clase tiene como objetivo representar un CDP y realizar operaciones sobre el mismo.

Esta clase hereda de la clase EntidadBancaria.

Esta clase tiene como atributos un cliente, un nombre, un apellido, un id, una fecha de creacion, una fecha de vencimiento, un CDPData, un monto, un periodo de pago, un interes, una moneda, un tipo de interes, un plazo y un periodo.
