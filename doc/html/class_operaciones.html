Descripción detallada

Clase que realiza las operaciones bancarias.

Esta clase realiza las operaciones bancarias como depositar, retirar, transferir, pagar prestamo, solicitar prestamo, consultar saldo, escribir historial, consultar movimientos y consultar prestamos.
